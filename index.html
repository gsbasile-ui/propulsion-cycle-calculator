Compressible Flow + Shock Tools

Conical shock needs a Taylor-Maccoll numerical solver. This page gives the honest reference note; use the VT calculator for conical-shock numbers.

Output

Isentropic: Tt/T = 1 + (gamma - 1)/2*M^2 Pt/P = (Tt/T)^(gamma/(gamma - 1)) A/A* = (1/M)*[(2/(gamma + 1))*(1 + (gamma - 1)/2*M^2)]^((gamma + 1)/(2*(gamma - 1))) Normal shock: M2^2 = [1 + (gamma - 1)/2*M1^2] / [gamma*M1^2 - (gamma - 1)/2] p2/p1 = 1 + 2*gamma/(gamma + 1)*(M1^2 - 1) Oblique shock: theta-beta-M relation solved numerically for weak shock by default. Fanno/Rayleigh: Perfect-gas star relations, useful for table-style checks.

Compressor / Fan

Output

Tt_out_s = Tt_in*pi^((gamma - 1)/gamma) Tt_out = Tt_in + (Tt_out_s - Tt_in)/eta Pt_out = pi*Pt_in Power = mdot*cp*(Tt_out - Tt_in)

Burner / Afterburner

Output

Main burner: f = (cp_out*Tt_out - cp_in*Tt_in)/(eta*h_PR - cp_out*Tt_out) Afterburner style, if existing f is nonzero: f_add = ((1 + f_prev)*(cp_out*Tt_out - cp_in*Tt_in))/(eta*h_PR - cp_out*Tt_out) Pt_out = pi_b*Pt_in

Turbine Power Balance

Output

Tt_out = Tt_in - cp_c*(Tc_out - Tc_in)/(eta_m*(1 + f)*cp_t) Tt_out_s = Tt_in - (Tt_in - Tt_out)/eta_t Pt_out = Pt_in*(Tt_out_s/Tt_in)^(gamma/(gamma - 1))

Nozzle Choking / Exit Velocity

Output

Pt_exit = pi_n*Pt_in critical = ((gamma + 1)/2)^(gamma/(gamma - 1)) If Pt_exit/Pamb >= critical: choked, M = 1 If not choked: M from Pt/P V = sqrt(2*eta_n*cp*Tt*(1 - (P_exit/Pt_exit)^((gamma - 1)/gamma)))

Turbojet Specific Performance

Output

Specific thrust = (1 + f + f_AB)*V9 - V0 TSFC = (f + f_AB)/specific_thrust Jet power = 0.5*((1 + f + f_AB)*V9^2 - V0^2) eta_th = jet_power / ((f*h_PR + f_AB*h_PR_AB)*1000) eta_p = specific_thrust*V0/jet_power

Split-Flow Turbofan

Output

F_fan = mdot_bypass*(V19 - V0) F_core = mdot_core*((1 + f)*V9 - V0) F_net = F_fan + F_core + pressure_thrust Jet power = 0.5*mdot_bypass*(V19^2 - V0^2) + 0.5*mdot_core*((1 + f)*V9^2 - V0^2) eta_th = jet_power/(mdot_core*f*Q_R*1000) eta_p = F_net*V0/jet_power TSFC = mdot_core*f/F_net
